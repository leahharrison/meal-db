## About The Project

The MealDB app is a simple React app that uses a public API to search for recipes.

### Built With

* React
* HTML
* CSS
* JavaScript


## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/leahharrison/meal-db.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the server
   ```sh
   npm start
   ```
